# Enterprise Dashboard with Real-Time Analytics

## Project Overview
This project is a modern enterprise dashboard built using React, TypeScript, Redux Toolkit, and Material UI. It displays analytics data using charts and provides simulated real-time updates.

## Features
- Dashboard cards showing Active Users, Sales, and Revenue
- User Growth Line Chart
- Monthly Sales Bar Chart
- Real-time dashboard updates
- Sidebar navigation layout
- Multiple pages using React Router
- Lazy loading using React.lazy for performance optimization

## Tech Stack
- React 18
- TypeScript
- Redux Toolkit
- React Router v6
- Material UI
- Recharts
- Vite
